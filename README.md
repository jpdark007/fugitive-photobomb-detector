# fugitive-photobomb-detector
"Où est Xavier ?" est une app Flutter 100% hors-ligne. Scannez les arrière-plans de vos photos via l'IA pour y chercher Xavier Dupont de Ligonnès, ou choisissez vos propres visages de référence pour traquer qui vous voulez ! Vos photos restent privées. Un "Où est Charlie" sur mesure.
